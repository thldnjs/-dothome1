The trigger for web
development

저는 어릴 때부터 대항해시대나 코에이 삼국지 같은 고전게임을
좋아했습니다. 그렇다보니 자연스럽게 컴퓨터에 앉아 있는 시
간이 많았고 직접 게임을 만들어보고 싶다는 생각이 들었습니
다. 간단한 게임 툴을 이용하여 직접 스크립트를 제작하고 변수
를 만들어 어떤 상황에 어떤 대사나 행동이 들어가야 하는지를
직접 고민해 본다는건 저 자신이 그 게임 속에 몰입하는 것만 같
아 새롭고 행복했습니다. 20살 때 대학에서 영상편집을 하면서
저는 컴퓨터로 하는 작업에 흥미가 있다는 것을 깨달았고 프리
렌서로 일을 하던 도중 친구가 선물해 준 프로그래밍 책을 읽게
되었습니다. 이후 저는 처음보는 컴퓨터 언어에 대해 많은 호기
심과 흥미를 갖게 되었고 언젠가는 좋은 회사에서 코딩으로 꿈
을 펼처 보고 싶다는 생각을 했습니다.
