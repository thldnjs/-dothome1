포트폴리오입니다.

제가 하고 싶은 포트폴리오를 조사해 보았습니다. 3번째 위주의 포트폴리오의
느낌으로 가지만 좀더 애니메이션이 다양한 사이트를 다시 리서치해서 오겠습니다.

첫번째 사이트

최근 레트로가 다시 유행을 타게 되면서 옛날의 감성을 좋아하는 사람들이 늘어나기 시작했다.
그래서인지 이런 느낌의 사이트에 관심을 갖게 되었다.
레트로 같은 느낌은 주되 너무 과하지 않은 효과들과 요소들을 사용한다면 좋을 것같다.

두번째 사이트

돌아가며 조명이 비춰지는 부분이 이쁘며 백그라운드에 사진을 넣어둠으로써 공연을 보는 듯한 느낌을 주는 깔끔하고 멋있는 사이트이다.
차용하기에는 좋지만 단점이라면 어두운 느낌이 많이 드는 효과가 많아서 밝은 분위기의 포트폴리오로는 부족할 것 같다.

세번째 사이트

심플하면서도 다양한 포트폴리오들을 넣을 수 있는 내용들이 많을 것 같아서 넣어두었다. 간단한 이펙트들이 많지만 알맞는 사진들을 잘 사용하여 집어 넣게 된다면
좋은 포트폴리오 영상이 될 수 있을 것 같다.

네번째 사이트

심플한 듯 하지만 효과가 정말 많이 들어가 있고 포트폴리오적인 느낌으로는 좋을 것 같다. 오른쪽으로 움직이는 스크롤바를 이동하고 클릭했을 때 자동으로 스크롤이 밑으로
내려가게 되며, 여러가지 포트폴리오를 보여준다.

다섯번째 사이트

글씨를 쓰는 듯한 효과가 정말 인상적이고 깔끔한 레이아웃을 통해 눈이 부담스럽지 않는 느낌을 준다. 포트폴리오를 만들게 된다면
글씨를 쓰는 듯한 효과를 정말 사용해보고싶다.

여섯번째 사이트

로고를 넣는 방식이나 다양한 메뉴들을 오른쪽으로 움직이는 스크롤바로 만들고 클릭 했을 경우 양 밑에 사이드로 캐릭터들이 나오는 효과는 정말 인상적이고
멋있다고 생각한다.

일곱번째 사이트

물건을 스크롤을 내릴때 돌아가게 하는 방식이나 밑에 글들이 서로 교차하며 반복적으로 움직이는 듯한 느낌이 마음에 듭니다.

여덞번째 사이트

일러스트를 이용한 간단하면서도 깔끔한 디자인으로 포트폴리오 용으로써는 정말 최고라고 생각합니다.

아홉번째 사이트

슬라이드 이펙트가 포트폴리오와 잘 어울리고 깔끔하고 부드러운 느낌을 주어 쇼핑몰 사이트로도 잘 표현된다고 생각합니다.

열번째 사이트

이미지에 효과가 들어가여 오버효과를 주었을 때 구겨지는 듯한 느낌이 깔끔하고 오버효과 시에는 동영상이 실행되는 느낌이 포트폴리오 용으로 좋다고 생각했습니다.
